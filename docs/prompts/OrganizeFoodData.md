# Task: Organize CSV Data Based on Gemini Analysis Markdown

## Objective

I have a CSV dataset and a Gemini-generated Markdown analysis document.

Your task is to use the Gemini Markdown document as the primary reference and reorganize the CSV data into a clean, structured format suitable for future use.

This task mainly involves tedious data processing and organization. Please focus on accuracy, consistency, and maintaining the logic defined by Gemini.

---

## Reference Priority

Please follow this priority order:

1. **Gemini Markdown file**

   * This is the authoritative source.
   * Follow its categories, structure, naming conventions, and logic.

2. **CSV file**

   * This is the original data source.
   * Extract, clean, transform, and reorganize information from it.

3. **DeepSeek Markdown file (optional reference only)**

   * Use it only if it provides additional useful information.
   * Do NOT override Gemini's structure or decisions.
   * If there is any conflict, always follow Gemini.

---

## Required Work

Please:

1. Analyze the Gemini Markdown structure:

   * Identify categories
   * Identify required fields
   * Identify data relationships
   * Identify formatting rules

2. Process the CSV:

   * Remove unnecessary noise
   * Normalize inconsistent formats
   * Fix obvious formatting issues
   * Map CSV fields into Gemini's structure

3. Generate organized output:

   * Keep the structure clean and maintainable
   * Use consistent naming
   * Avoid unnecessary duplication
   * Preserve important original information

4. Before finalizing:

   * Check for missing important data
   * Check category consistency
   * Check whether output follows Gemini's logic
   * Report any uncertain cases instead of making arbitrary assumptions

---

## Output Requirements

Please provide:

1. Final organized data file(s)

   * Preferred format: Markdown / JSON / CSV depending on the project structure

2. A short processing report:

   * What was cleaned
   * What was transformed
   * Any assumptions made
   * Any unresolved issues

Do not redesign the content logic. Your role is mainly data organization, transformation, and quality control.
